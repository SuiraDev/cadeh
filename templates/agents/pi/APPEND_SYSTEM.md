# CADEH — System prompt append

Siga as instruções do `AGENTS.md` na raiz do projeto.

Use os prompt templates abaixo para orquestrar o fluxo de engenharia de software:
- `/cadeh-continue` — retomar sessão
- `/cadeh-spec` — especificar feature (SDD)
- `/cadeh-plan` — planejar implementação
- `/cadeh-tasks` — quebrar em tarefas
- `/cadeh-implement` — implementar uma T-xx
- `/cadeh-memory` — salvar contexto
- `/cadeh-workflow` — mapa completo

CodeGraph: use ferramentas MCP para navegar no código (`.codegraph/`).
