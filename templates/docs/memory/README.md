# Memória narrativa (por feature)

Complementa `.cadeh/state.yml` com decisões, bloqueios e log de sessão.

| Artefato | Função |
|----------|--------|
| `.cadeh/state.yml` | Feature, fase, T-xx, notas curtas |
| `docs/memory/<slug>.md` | Contexto humano entre chats |
| CodeGraph | Código — **não** duplique paths/símbolos aqui |

Criado automaticamente por `cadeh new feature <slug>`.

Cursor: `/cadeh-memory` ao encerrar · `/cadeh-continue` ao retomar.
